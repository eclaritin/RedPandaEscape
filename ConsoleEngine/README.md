# ConsoleEngine (short overview)

## Organized into 3 namespaces

1. `ConsoleEngine.Data` stores classes used to represent standalone 'things' in the game (like tiles & items) or raw information (like vectors).
2. `ConsoleEngine.Properties` stores classes that are primarily used to add properties & behavior to other objects.
3. `ConsoleEngine.Services` stores static classes that are the main entrypoint into the game engine.
